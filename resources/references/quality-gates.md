# Quality Gates for SGEO Audits

<!-- Updated: 2026-03-12 -->

## Content Minimums

| Page Type | Min Words | Unique Content % | Title Length | Meta Desc Length |
|-----------|-----------|-------------------|-------------|-----------------|
| Homepage | 1,000 | 80% | 50-60 chars | 150-160 chars |
| Product/Service | 800 | 85% | 50-60 chars | 150-160 chars |
| Blog Post | 1,500 | 90% | 50-60 chars | 150-160 chars |
| Tool/Feature Page | 600 | 85% | 50-60 chars | 150-160 chars |
| Category Page | 300 | 70% | 50-60 chars | 150-160 chars |
| About Page | 500 | 95% | 50-60 chars | 150-160 chars |
| Landing Page | 500 | 80% | 50-60 chars | 150-160 chars |

## Heading Requirements

- Exactly 1 H1 per page
- H1 contains primary keyword
- H2s used for major sections (2-8 per page)
- No skipped heading levels (H1→H3 without H2)
- Headings are descriptive, not generic ("Our Services" → "AI Photo Editing Services")

## Image Requirements

- All images have descriptive alt text
- Alt text ≤ 125 characters
- Decorative images use alt=""
- Images have explicit width/height (CLS prevention)
- Lazy loading on below-fold images
- WebP or AVIF format preferred

## Internal Linking

- Minimum 3 internal links per page
- No orphan pages (≤1 incoming link)
- Descriptive anchor text (not "click here")
- No more than 100 links per page

## Schema Requirements

- Valid JSON-LD syntax (no errors)
- No placeholder text ("Your Company", "example.com")
- No deprecated types (HowTo)
- Required properties present for each type
- sameAs links: ≥2 for SEO, ≥5 for GEO

## AI Citability Minimums (GEO)

- ≥3 statistical anchors per page
- ≥1 self-contained passage (134-167 words)
- ≥1 direct answer pattern per page
- FAQ section on key pages (≥3 Q&A pairs)
